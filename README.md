# ErgoSafe Reborn

Reconstructed and upscaled OHS application with Melly AI, 3D Training, and Analytics.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)

### Installation
```bash
# 1. Provide permissions if on Windows (needed for scripts)
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

# 2. Install dependencies
npm install
```

### Running the App
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🌟 Features implemented
- **Dark Navy Premium Theme** using Tailwind CSS.
- **Melly AI Agent** with pulsing voice animation.
- **Interactive 3D Training** using React Three Fiber.
- **Glassmorphism UI** for cards and dashboard.
- **Animated Charts** using Framer Motion.

## ⚠️ Notes
- **Ads.txt** is located in `public/ads.txt`.
- **Melly Avatar** missing? Place your `Melly_Avatar.png` in `src/assets/`.
